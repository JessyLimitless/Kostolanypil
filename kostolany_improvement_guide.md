## 1순위: 즉시 반영 (필수)

### 1-1. 사례 반복 정리

**문제:** 동일한 사례·명언이 여러 챕터에 중복 등장하여 독자가 지루함을 느낌.

**중복 목록:**

| 사례/명언 | 등장 위치 | 처리 방법 |
|-----------|----------|----------|
| 아이젠하워 심장마비 사건 | 1장(상세), 2장(상세), 6장(언급) | **1장에서만 상세 서술.** 2장은 "1장에서 살펴본 아이젠하워 사건이 보여주듯" 한 줄 참조로 대체. 6장도 동일하게 축약. |
| IMF·금융위기·코로나 3회 비교 | 프롤로그(간략), 3장(상세 표), 7장(상세 표) | **3장에서 상세 표로 1회만 제시.** 프롤로그는 "세 번의 위기 모두 공포 속 매수자가 이겼다" 수준으로 축약. 7장은 "3장에서 확인한 세 번의 사이클을 역발상 관점에서 다시 보면..." 으로 전환하되, 동일한 표를 반복하지 않고 새로운 열(공포 내러티브, 바닥 매수 수익률)만 추가. |
| "수면제를 먹고 10년을 자라" | 프롤로그, 2장, 5장, 10장 | **2장에서만 원문 인용.** 프롤로그·5장·10장은 "코스톨라니의 유명한 수면제 조언처럼" 한 줄 참조로 대체. |
| 독일 국채 140배 수익 | 1장(상세), 2장(언급), 7장(상세) | **1장에서 상세 서술.** 7장은 "1장에서 다룬 독일 국채 사례의 논리 구조를 분석하면..." 으로 전환하여 반복 서술 없이 새로운 관점(역발상 4단계 프로세스) 제시. |
| "정보는 대부분 무의미하다" | 프롤로그, 2장, 6장 | **2장에서만 상세 인용.** 프롤로그는 한 줄 언급, 6장은 "2장에서 다룬 코스톨라니의 정보관처럼" 참조. |
| 60대 여성 투자자 사례 | 프롤로그, 5장 | **프롤로그에서만 서술.** 5장은 "프롤로그에서 언급한 60대 여성 투자자 사례가 이를 증명한다" 한 줄 참조. |

**예상 효과:** 2,000~3,000자 감소, 읽기 밀도 상승, 챕터 간 연결감 강화.

### 1-2. 면책 조항 추가

**위치:** 프롤로그 앞 또는 부록 D 뒤에 별도 섹션으로 삽입.

**삽입할 텍스트:**

```
## 면책 조항

본 책의 내용은 정보 제공 및 교육 목적으로 작성되었으며, 
특정 금융 상품이나 투자 전략에 대한 권유 또는 추천이 아닙니다.

본 책에서 언급된 종목명, 자산 비중, 시장 진단은 코스톨라니의 투자 철학을 
설명하기 위한 예시이며, 특정 시점의 매수·매도를 권유하는 것이 아닙니다.

모든 투자 결정과 그에 따른 결과는 독자 본인의 판단과 책임입니다.
투자에는 원금 손실의 위험이 있으며, 과거 수익률이 미래 수익을 보장하지 않습니다.

중요한 투자 결정을 내리기 전에 공인된 재무 상담사와 상의하시기 바랍니다.
```

### 1-3. 인용문(blockquote) 렌더링 수정

**문제:** 코스톨라니 명언 blockquote 위에 빈 큰따옴표 블록(`> "`)이 불필요하게 표시됨.

**발생 위치:** 프롤로그, 1장, 8장 등 명언 인용부 전체.

**수정 방법:** HTML에서 아래 패턴을 찾아 제거:

```html
<!-- 제거 대상 -->
<blockquote>
  <p>"</p>
</blockquote>

<!-- 또는 MD 소스에서 -->
> "
```

명언 blockquote만 남기고, 그 위의 빈 따옴표 블록을 삭제할 것.

## 2순위: 콘텐츠 가치 상승 (강력 권장)

### 2-1. 프롤로그에 집필 동기 1~2문장 추가

**목적:** 독자가 "왜 이 저자가 코스톨라니를 한국에 소개하려 하는가"를 이해하게 함. 긴 경험담이 아니라 최소한의 맥락만 제공.

**위치:** 프롤로그 "이 책이 당신에게 주는 것" 섹션 첫 문장 앞.

**삽입할 텍스트 (1~2문장):**

```
10년간 데이터 기반 컨설팅을 해온 사람으로서, 
코스톨라니의 체계적 사고 — 금리 사이클로 시장을 읽고, 
심리를 수치화하여 판단하는 접근법 — 에 깊이 공감했다. 
이 책은 그 공감을 한국 시장의 언어로 옮긴 결과물이다.
```

**주의:** 이 이상의 개인 이야기는 넣지 않는다. 이 책의 주인공은 코스톨라니이지 저자가 아니다. 저자의 역할은 해설자이며, 해설자는 무대 뒤에 있어야 한다.

### 2-2. 코스톨라니 원전 인용 대조

**대상:** 한국어 번역본 『돈, 뜨겁게 사랑하고 차갑게 다루어라』 (미래의창 출판) 기준.

**주요 검증 항목:**

- [ ] 10가지 권고사항 문구 (6장)
- [ ] 10가지 금기사항 문구 (6장)
- [ ] "2×2=5-1" 비유의 정확한 표현 (2장)
- [ ] 달걀 모형의 구간 설명 — A~F 각 구간의 원전 설명과 일치 여부 (3장)
- [ ] "고통 자금(Zittergeld)" 개념 설명 (2장)
- [ ] 소신파/부화뇌동파의 원전 용어 확인 (5장)
- [ ] "주식시장의 90%는 심리학" 인용 정확도 (프롤로그, 8장)
- [ ] 백만장자 정의 인용 정확도 (11장)

**처리 방법:** 원전과 다른 표현이 발견되면 원전 문구로 수정하거나, "코스톨라니의 표현을 재구성하면" 식으로 간접 인용으로 전환.

### 2-3. 9장 현재 시점 데이터 보강

**문제:** 2026년 3월 시점 진단 챕터인데, 정작 "지금 여기"를 느끼게 하는 실시간 데이터가 부족.

**추가할 데이터 (9-3절 "자산별 비중 판단 프레임워크"에 삽입):**

```
2026년 3월 현재 주요 지표:

- 한국은행 기준금리: 2.50% (2025.5 이후 6회 동결)
- 코스피 PBR: ___배 (역사적 밴드 ___~___배 중 ___위치)
- M2 통화량 전년비 증가율: ___%
- 투자자 예탁금: ___조 원
- VKOSPI: ___
- 신용잔고: ___조 원
- 원달러 환율: ___원
- 개인 순매수/순매도 최근 1개월: ___
```

→ 이 데이터를 유동성+심리 매트릭스에 직접 대입하여 "현재 시나리오 ②(유동성↑ 심리↓)로 판단하는 근거"를 수치로 보여줄 것.

**데이터 소스:** 한국은행 경제통계시스템(ECOS), 한국거래소(KRX), 금융투자협회, 네이버 금융.

## 3순위: 프리미엄급 완성 (선택)

### 3-1. 부록 C 인터랙티브 워크시트

**현재:** 정적 텍스트로 된 점수 입력 표.

**목표:** 독자가 직접 점수를 입력하면 자동으로 결과(시나리오 번호 + 행동 가이드)가 표시되는 웹 폼.

**구현 방향:**
- Step 1~3 각 질문에 라디오 버튼 또는 셀렉트박스
- 점수 자동 합산
- Step 4에서 유동성 점수 × 심리 점수로 4가지 시나리오 매트릭스 자동 매칭
- 결과 화면: "현재 당신의 판단: 시나리오 ②(최적 매수 구간)" + 해당 행동 가이드 표시
- 결과를 이미지 또는 텍스트로 저장/공유 기능 (선택)

---

### 3-2. 표지 디자인

**용도:** EPUB/PDF 배포용 + 웹 OG 이미지.

**디자인 방향:**
- 다크 네이비 배경 (#1B4F72)
- 중앙에 달걀 모형 골드 라인 일러스트
- 상단: 책 제목 (화이트, 세리프 폰트)
- 하단: 부제 (골드, 산세리프)
- 코스톨라니 실루엣 또는 증권거래소 빈티지 이미지 (선택)

**제작 도구:** Canva Pro 또는 Midjourney + Photoshop.

### 3-3. 핵심 차트 2~3개 추가

**차트 후보 (택 2~3):**

**① 한국 금리 vs 코스피 오버레이 차트 (1997~2026)**
- X축: 연도, Y축 좌: 기준금리(%), Y축 우: 코스피
- 달걀 국면 A~F를 배경색으로 표시
- → 3장 또는 부록 B에 삽입

**② 삼성전자 PBR 밴드 차트 (2015~2026)**
- PBR 상단(2.5배)·중앙(1.5배)·하단(1.0배) 밴드 + 실제 PBR 추이
- 2024.7 고점, 2024.11 저점, 2026.1 고점 포인트 표시
- → 4장에 삽입

**③ 유동성+심리 매트릭스 4사분면 차트**
- X축: 심리(비관←→낙관), Y축: 유동성(부족←→풍부)
- 각 사분면에 시나리오 번호 + 한국 시장 사례 연도 표시
- 현재 위치를 별표로 표시
- → 8장에 삽입

**제작 도구:** Recharts(React), D3.js, 또는 정적 SVG.
